Steps for making ruby web app using sinatra:
1. Install the gems--> using bundle install
2. use the gems in your main file here theweb.rb
3. run the file using "ruby file.rb"
4. Make use of templates, you can use erb, yml etc, here we use erb.
5. keep all the views in your View folder.
6. You can make use of layouts for common code.
7. Add Bootstrap template for layout
